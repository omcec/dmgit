# Modo IE no Edge

O suporte para ActiveX foi descontinuado no Edge, o que pode ser o motivo da dificuldade que está enfrentando. No entanto, há algumas soluções que podemos tentar:

1. Habilitar o modo Internet Explorer no Edge

O novo Edge tem um modo Internet Explorer incorporado que pode ajudar a resolver seu problema. Aqui estão os passos para ativá-lo:

* Digite "edge://flags" (sem aspas) na barra de endereços do Edge e pressione Enter.
* Pesquise "Enable IE Integration" ou "Habilitar integração com o IE" na barra de pesquisa.
* Selecione "IE Mode" no menu suspenso.
* Reinicie o Edge.
* Agora, ao tentar acessar um site que requer Internet Explorer, o Edge deve abrir esse site no modo IE.

2. Use a política de grupo para configurar o modo IE no Edge

Se você tiver acesso de administrador, pode usar a política de grupo para configurar o modo IE. Este é um processo mais complexo e recomendo que seja feito por um profissional de TI.

Se nenhuma dessas soluções funcionar, talvez seja necessário continuar usando o Internet Explorer para acessar esse site específico, ou entrar em contato com o administrador da rede para ver se existe alguma atualização ou alternativa que funcione melhor com navegadores modernos.

Espero que isso ajude! Se você tiver mais problemas ou perguntas, por favor me avise.
