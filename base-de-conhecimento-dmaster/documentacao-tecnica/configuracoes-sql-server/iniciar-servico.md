# Iniciar serviço

Para iniciar o serviço do SQL é preciso seguir os passos abaixo.

Acessar o **SQL Server Configuration Manager.**

![](https://dmaster.gitbook.io/base-de-conhecimento-dmaster/~gitbook/image?url=https%3A%2F%2F981618378-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FaHl55K4HG6CuWy9zUjR9%252Fuploads%252F8btDNt5dp7GjXUw2F1gk%252Fsql%2520server%2520configuration.jpg%3Falt%3Dmedia%26token%3D561723ec-f9af-40f3-a8d0-80d14b0c833f\&width=768\&dpr=4\&quality=100\&sign=6f6c0a92\&sv=2)

Clicar em **Serviços do SQL Server** e ao lado clicar com botão direito em **SQL Server(MSSQLSERVER)** e clicar em **Iniciar.**

![](https://dmaster.gitbook.io/base-de-conhecimento-dmaster/~gitbook/image?url=https%3A%2F%2F981618378-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FaHl55K4HG6CuWy9zUjR9%252Fuploads%252FP2xeNUJS6UWlVwUBADQ0%252Finiciar%2520servico.jpg%3Falt%3Dmedia%26token%3D11e9e301-b1c6-4ffe-9d29-f5f7637f6df7\&width=768\&dpr=4\&quality=100\&sign=17f990a\&sv=2)
