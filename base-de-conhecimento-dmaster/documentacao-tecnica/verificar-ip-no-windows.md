# Verificar IP no Windows

#### omo verificar IP do computador ? <a href="#como-verificar-ip-do-computador" id="como-verificar-ip-do-computador"></a>

Sabemos que IP é um número que identifica o computador na rede, sendo assim, este número deve ser válido, para que o computador tenha acesso a internet e esteja acessível na rede. Abaixo será mostrado o passo a passo de como verificar o IP do computador no Windows.

O primeiro passo é acessar o **painel de controle**, alterar a exibição para **ícones grandes** e clicar em **central de rede e compartilhamento**.

![](https://dmaster.gitbook.io/base-de-conhecimento-dmaster/~gitbook/image?url=https%3A%2F%2F981618378-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FaHl55K4HG6CuWy9zUjR9%252Fuploads%252FO6qiJT6IEb2jjgexCb96%252Fcentral%2520de%2520rede%2520e%2520compartilhamento.PNG%3Falt%3Dmedia%26token%3Dd4f22670-1d4b-47c4-8cd1-b3d53883f20a\&width=768\&dpr=4\&quality=100\&sign=74521b47\&sv=2)

Na próxima tela, clicar em **alterar as configurações do adaptador**.

![](https://dmaster.gitbook.io/base-de-conhecimento-dmaster/~gitbook/image?url=https%3A%2F%2F981618378-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FaHl55K4HG6CuWy9zUjR9%252Fuploads%252F6edTKOwRncwsJ6bRtAqC%252FAlterar%2520configuracoes%2520do%2520adaptador.PNG%3Falt%3Dmedia%26token%3D2f9148a4-71ec-4981-9e05-a7287073c9dc\&width=768\&dpr=4\&quality=100\&sign=568f81ec\&sv=2)

Na tela de **conexões de rede**, utilizar o duplo click do mouse no registro da internet que o computador está conectado.

![](https://dmaster.gitbook.io/base-de-conhecimento-dmaster/~gitbook/image?url=https%3A%2F%2F981618378-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FaHl55K4HG6CuWy9zUjR9%252Fuploads%252Fb9uFecamM5vE7BnQ2o06%252FRedes%2520de%2520internet.PNG%3Falt%3Dmedia%26token%3D3ac8c0b4-8896-4d34-9678-40bb09237336\&width=768\&dpr=4\&quality=100\&sign=4170995f\&sv=2)

Na tela de status, clicar em **detalhes** e na próxima tela, no campo **Endereço IPv4** será mostrado o **IP** daquele computador.

![](https://dmaster.gitbook.io/base-de-conhecimento-dmaster/~gitbook/image?url=https%3A%2F%2F981618378-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FaHl55K4HG6CuWy9zUjR9%252Fuploads%252FSezDAto8zT4sALDDEKWI%252FDetalhes.PNG%3Falt%3Dmedia%26token%3D57e85ec6-bb32-4f45-a83e-d19e8022c078\&width=768\&dpr=4\&quality=100\&sign=d2037ddd\&sv=2)![](https://dmaster.gitbook.io/base-de-conhecimento-dmaster/~gitbook/image?url=https%3A%2F%2F981618378-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FaHl55K4HG6CuWy9zUjR9%252Fuploads%252F0VfAr3xRceZ6qBILopHi%252FIP.PNG%3Falt%3Dmedia%26token%3Da8dd3fe5-8ccc-4e49-817a-d0a9a8e7fc9d\&width=768\&dpr=4\&quality=100\&sign=b953f34d\&sv=2)

{% hint style="info" %}
Caso o IP mostrado seja começado em 169, significa que Windows solicitou um endereço IP e não o recebeu. Quando isso ocorre, o computador ficará sem rede e possivelmente sem internet.&#x20;

Para correção do IP, é preciso reiniciar o aparelho da internet e computador para que o Windows consiga receber um IP válido.
{% endhint %}
