# Cote Fácil

**Cote Fácil** é um empresa de cotações de produtos e o sistema realiza o processo de gerar um arquivo para que o usuário possa enviá-lo para essa empresa.

A seguir, será explicado o processo para exportação do arquivo pelo sistema DMASTER.

Para exportar toda a base de produtos do sistema DMASTER, é preciso acessar no DMASTER o módulo **Utilitários > Atualizar > Cote Fácil.**

![](https://dmaster.gitbook.io/base-de-conhecimento-dmaster/~gitbook/image?url=https%3A%2F%2F981618378-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FaHl55K4HG6CuWy9zUjR9%252Fuploads%252FvBtQeYPBums7Xq7Xf50c%252FMenu%2520cote%2520facil.png%3Falt%3Dmedia%26token%3D4f6b0b74-9992-466f-ab0d-4213239ffad4\&width=300\&dpr=4\&quality=100\&sign=a0eef0cb\&sv=2)

Ao acessar o módulo, será gerada a mensagem de exportação, sendo necessário clicar em **Sim.**

![](https://dmaster.gitbook.io/base-de-conhecimento-dmaster/~gitbook/image?url=https%3A%2F%2F981618378-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FaHl55K4HG6CuWy9zUjR9%252Fuploads%252FBYlRVtPqz4JF7pTe3YGn%252Fconfirmacao.PNG%3Falt%3Dmedia%26token%3D851b6f4c-8c88-4e3b-9583-2588e4886885\&width=300\&dpr=4\&quality=100\&sign=8d77eb52\&sv=2)

Após a exportação, o arquivo será gerado na pasta **MBHSist > Cote Fácil.**

![](https://dmaster.gitbook.io/base-de-conhecimento-dmaster/~gitbook/image?url=https%3A%2F%2F981618378-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FaHl55K4HG6CuWy9zUjR9%252Fuploads%252FHexdZGQ55TmPKxxtxJn0%252FArquivos%2520exportado.png%3Falt%3Dmedia%26token%3D1349f53e-ea7a-401f-a786-f1427bf55d53\&width=300\&dpr=4\&quality=100\&sign=6830a09a\&sv=2)

Com o arquivo exportado, o usuário deverá realizar o envio para o Cote Fácil através do site da empresa [https://cotefacil.online/](https://cotefacil.online/).

A DMASTER não tem acesso e informações quanto ao processo de envio do arquivo para o Cote Fácil.

Além da possibilidade de exportar um arquivo com toda a base de dados do sistema, também é possível gerar um arquivo apenas com produtos selecionados atavés do módulo de manutenção de pedidos do DMASTER FARMA. O processo será explicado a seguir.

Acessar o módulo **Relatórios > Manutenção de pedidos.**

![](https://dmaster.gitbook.io/base-de-conhecimento-dmaster/~gitbook/image?url=https%3A%2F%2F981618378-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FaHl55K4HG6CuWy9zUjR9%252Fuploads%252Fm3ZT5byk3jp0Td7H0X3d%252FManutencao%2520de%2520pedidos.png%3Falt%3Dmedia%26token%3Ddefda785-206f-46c7-9ac3-dcf6472a1e8c\&width=300\&dpr=4\&quality=100\&sign=e8f39e4d\&sv=2)

Na tela de **manutenção de pedidos**, após selecionar o pedido desejado, escolher o **tipo de saída Cote Fácil** e o sistema irá gerar o arquivo na pasta **MBHSist > Cote Fácil.**
