# DM Object Pro

**DM ObjectPro** tem o objetivo de fidelizar clientes através de descontos em produtos fornecidos por uma Rede. O aplicativo possui integração com o Dmaster Farma e ao realizar uma venda no mesmo, ela é gerada automaticamente no sistema Dmaster para ser finalizada.

O aplicativo pode ser instalado no servidor e terminais da loja.

Para abrir o DM ObjectPro, o sistema Dmaster Farma já deve estar iniciado no dia corrente

Os processos para configuração e utilização do aplicativo serão explicados a seguir:

#### Copiando arquivos <a href="#copiando-arquivos" id="copiando-arquivos"></a>

Copiar todos os arquivos referente ao **DMObjectPro** do **Servidor Dmaster** para a pasta **MBHSist** do **cliente**.

#### Configurando o aplicativo <a href="#configurando-o-aplicativo" id="configurando-o-aplicativo"></a>

Com o executável atualizado, abrir o **DMConfigurações** e clicar em **Object Pro**.

![](https://dmaster.gitbook.io/base-de-conhecimento-dmaster/~gitbook/image?url=https%3A%2F%2F981618378-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FaHl55K4HG6CuWy9zUjR9%252Fuploads%252FFGZfa63aXVPJxXQCf5Pz%252Fdmconfig.png%3Falt%3Dmedia%26token%3D0c2742af-37af-4b27-8118-6ec0da05932f\&width=300\&dpr=4\&quality=100\&sign=4b7857fc\&sv=2)

Ao clicar no botão **Object Pro**, será aberta a tela para configuração da **URL** e **tipo de integração** do aplicativo.

A informação de URL é padrão para a **Rede Soma**. Caso o cliente pertença a outra Rede, o mesmo deve informar qual URL será utilizada.

O tipo de integração utilizado será **Desconto**. Único tipo implementado até então.

Após realizar os passos acima, clicar em **Gravar** e será exibida a mensagem de confirmação.

![](https://dmaster.gitbook.io/base-de-conhecimento-dmaster/~gitbook/image?url=https%3A%2F%2F981618378-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FaHl55K4HG6CuWy9zUjR9%252Fuploads%252Fu6gznvrWqre66Agf3y1c%252FConfiguracao.PNG%3Falt%3Dmedia%26token%3D28d63728-0d5e-4d44-a9af-789736c21328\&width=300\&dpr=4\&quality=100\&sign=de45cbb\&sv=2) ![](https://dmaster.gitbook.io/base-de-conhecimento-dmaster/~gitbook/image?url=https%3A%2F%2F981618378-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FaHl55K4HG6CuWy9zUjR9%252Fuploads%252FlvHJLUduQh4zFXT0oZCR%252FMensagem%2520configuraca.PNG%3Falt%3Dmedia%26token%3Dc92b1b5a-2f61-49e1-93b6-c8a896ed4cfa\&width=300\&dpr=4\&quality=100\&sign=d907744b\&sv=2)

Caso o aplicativo for instalado no servidor e terminais, este passo deve ser realizado apenas uma vez no servidor, para que as informações de configuração sejam inseridas no banco de dados.

#### Iniciando o aplicativo <a href="#iniciando-o-aplicativo" id="iniciando-o-aplicativo"></a>

Com as configurações concluídas, ao iniciar o aplicativo será aberta a tela para login. O atendente deverá inserir **usuário** e **senha** para ter acesso ao aplicativo.

Após inserir usuário e senha corretos, utilizar o botão **\[F2] Entrar** ou a tecla **F2** e a tela inicial do aplicativo será aberta.

![](https://dmaster.gitbook.io/base-de-conhecimento-dmaster/~gitbook/image?url=https%3A%2F%2F981618378-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FaHl55K4HG6CuWy9zUjR9%252Fuploads%252F7SiWd4p1npoEtnFl5qVr%252FUsuario.PNG%3Falt%3Dmedia%26token%3D997e813e-213c-4966-ba93-6fda04af570a\&width=300\&dpr=4\&quality=100\&sign=9afd24b5\&sv=2)

O cadastro do atendente no Dmaster Farma deve obrigatoriamente conter CPF para que o mesmo consiga fazer login.

Caso o atendente **NÃO** tenha CPF informado no cadastro de usuário do Dmaster Farma, a seguinte mensagem será exibida.

![](https://dmaster.gitbook.io/base-de-conhecimento-dmaster/~gitbook/image?url=https%3A%2F%2F981618378-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FaHl55K4HG6CuWy9zUjR9%252Fuploads%252F1Gd595g8w37qZXd7mwO0%252Fatendente%2520sem%2520cpf.PNG%3Falt%3Dmedia%26token%3Dc53ca3cb-ee24-4324-b42e-e3f15b3d869b\&width=300\&dpr=4\&quality=100\&sign=3dbf2202\&sv=2)

#### Cadastrando cliente no aplicativo <a href="#cadastrando-cliente-no-aplicativo" id="cadastrando-cliente-no-aplicativo"></a>

Após fazer login, é necessário cadastrar o cliente que está realizando a compra. Para isso, é necessário inserir o **CPF** do mesmo no campo **Pesquisa do cliente** e clicar no botão **Localizar** ou utilizar a **tecla F2**.

![](https://dmaster.gitbook.io/base-de-conhecimento-dmaster/~gitbook/image?url=https%3A%2F%2F981618378-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FaHl55K4HG6CuWy9zUjR9%252Fuploads%252FmgIsiUc14MmVQEqJSvL8%252FPesquisa%2520de%2520cliente.PNG%3Falt%3Dmedia%26token%3D1dc8d917-42f9-4a90-a9d0-d6504a06908f\&width=300\&dpr=4\&quality=100\&sign=59ead142\&sv=2)

Após inserir o CPF, será aberta a tela de cadastro para que sejam preenchidas as informações necessárias: **número do cartão** e **sexo (feminino ou masculino)**.

Com as informações preenchidas, utilizar o botão **Enviar cliente** ou a **tecla F2** para que as informações sejam gravadas.

Para cadastro do cliente no aplicativo, o mesmo já deve estar cadastrado com CPF no Dmaster Farma.

![](https://dmaster.gitbook.io/base-de-conhecimento-dmaster/~gitbook/image?url=https%3A%2F%2F981618378-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FaHl55K4HG6CuWy9zUjR9%252Fuploads%252FmbvPKvgliqJmgFrzoO0l%252Fimage.png%3Falt%3Dmedia%26token%3D89a650db-fa9a-4799-98a2-6e33dd6a0041\&width=300\&dpr=4\&quality=100\&sign=9e5dd185\&sv=2)

Caso o cliente **NÃO** esteja cadastrado no **Dmaster Farma** ou o cadastro esteja **SEM CPF informado**, ao digitar o CPF e clicar em **localizar** é exibida a seguinte mensagem.

![](https://dmaster.gitbook.io/base-de-conhecimento-dmaster/~gitbook/image?url=https%3A%2F%2F981618378-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FaHl55K4HG6CuWy9zUjR9%252Fuploads%252Fpk91KaaSUtBDERsBbCf3%252FCliente%2520nao%2520encontrado.PNG%3Falt%3Dmedia%26token%3Da3e7ccce-105c-485c-a8fe-e3a52f6502fc\&width=300\&dpr=4\&quality=100\&sign=806101ad\&sv=2)

#### Pesquisando cliente <a href="#pesquisando-cliente" id="pesquisando-cliente"></a>

Caso o cliente já seja cadastrado no Dmaster Farma e no aplicativo de fidelidade, é necessário apenas informar o **CPF** e o nome será carregado automaticamente.

![](https://dmaster.gitbook.io/base-de-conhecimento-dmaster/~gitbook/image?url=https%3A%2F%2F981618378-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FaHl55K4HG6CuWy9zUjR9%252Fuploads%252F7E6p48J5v9wsxQlG7epL%252FPesquisa%2520cliente.PNG%3Falt%3Dmedia%26token%3D9900c669-ea86-46d1-8301-875c421e7fbf\&width=300\&dpr=4\&quality=100\&sign=8518bfda\&sv=2)

#### Realizando venda <a href="#realizando-venda" id="realizando-venda"></a>

Após localizar o cliente, será necessário fazer o lançamento do(s) produto(s) da compra. Para isso, é necessário pesquisar o produto por **código interno**, **código de barras** ou **descrição** através do campo **Código**.

Após a busca pelo produto, serão carregadas as informações do mesmo e então será preciso inserir a quantidade desejada, clicar no botão **Incluir item** ou utilizar a **tecla F4.**

![](https://dmaster.gitbook.io/base-de-conhecimento-dmaster/~gitbook/image?url=https%3A%2F%2F981618378-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FaHl55K4HG6CuWy9zUjR9%252Fuploads%252FDo0MU3CtP3XS36j7b470%252FVenda.PNG%3Falt%3Dmedia%26token%3D6db753c4-5431-4882-b727-da06000d0180\&width=300\&dpr=4\&quality=100\&sign=9092edb8\&sv=2)

Ao Incluir item, o produto será carregado no grid com as seguintes informações: **descrição do produto**, **fabricante**, **quantidade do produto na venda**, **preço bruto**, **valor de** **desconto**, **preço de venda**, **valor total do produto** e **origem do desconto (Drogaria ou Rede).**

![](https://dmaster.gitbook.io/base-de-conhecimento-dmaster/~gitbook/image?url=https%3A%2F%2F981618378-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FaHl55K4HG6CuWy9zUjR9%252Fuploads%252F9bk2miG39P87HMz6Mon6%252FItem%2520lancado.PNG%3Falt%3Dmedia%26token%3D01469b44-4b7c-4e50-a84c-5d9763db6084\&width=300\&dpr=4\&quality=100\&sign=a5ddf980\&sv=2)

Com o produto carregado, agora é necessário consultar descontos, ou seja, verificar qual desconto é melhor para o cliente, o desconto da Drogaria ou da Rede. Para isso, utiliza-se a tecla **F5.** Após consultar os descontos, será exibida a mensagem como mostrada abaixo.

![](https://dmaster.gitbook.io/base-de-conhecimento-dmaster/~gitbook/image?url=https%3A%2F%2F981618378-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FaHl55K4HG6CuWy9zUjR9%252Fuploads%252FqkbGKlPBWI4UP2kWv5Ue%252FConsulta%2520desconto.PNG%3Falt%3Dmedia%26token%3D99db8548-5d6f-407d-a5e6-230a0d80b81d\&width=300\&dpr=4\&quality=100\&sign=566b9f17\&sv=2)

O desconto da Rede é aplicado apenas se o produto possuir código de barras válido no cadastro do mesmo no Dmaster Farma, ou seja, caso o produto possua desconto pela Rede, mas o cadastro esteja sem código de barras ou com código de barras incorreto, o desconto da Rede não será aplicado.

Caso o processo de consultar descontos já tenha sido realizado e seja necessário adicionar um novo produto, será preciso realizar o processo de consulta de descontos novamente após o novo produto ser adicionado.

#### Finalizando venda <a href="#finalizando-venda" id="finalizando-venda"></a>

Após os passos anteriores, é necessário finalizar a venda. Para isso, utiliza-se a tecla **F6** e então será aberta a tela para selecionar a forma de pagamento. Nesta tela é possivel selecionar as formas de pagamento dinheiro ou cartão (crédito e débito), de acordo com o que foi cadastrado no Dmaster Farma.

Após selecionar a forma de pagamento, caso a venda seja de entrega, é necessário selecionar o parâmetro **Venda c/ entrega**.

Para vendas de entrega que possuem frete, o mesmo deve ser lançado como produto, assim como é lançado no Dmaster Farm&#x61;**.**

![](https://dmaster.gitbook.io/base-de-conhecimento-dmaster/~gitbook/image?url=https%3A%2F%2F981618378-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FaHl55K4HG6CuWy9zUjR9%252Fuploads%252FKHsJ9Lm92Da2WKJ2AhXw%252FFinalizando%2520venda.PNG%3Falt%3Dmedia%26token%3D9eef6eb0-87d3-49e5-bc37-027b7334a5ec\&width=300\&dpr=4\&quality=100\&sign=803e6847\&sv=2)

Caso seja selecionada a forma de pagamento **dinheiro** e marcado o parâmetro **Venda c/ entrega**, é necessário informar o **valor a receber** para que seja calculado o troco, caso houver.

![](https://dmaster.gitbook.io/base-de-conhecimento-dmaster/~gitbook/image?url=https%3A%2F%2F981618378-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FaHl55K4HG6CuWy9zUjR9%252Fuploads%252Fg2ApomRmUo4nHQm1Vv9e%252FTroco.PNG%3Falt%3Dmedia%26token%3D946f4715-60e2-4fb5-bc46-b382a1f1f7f8\&width=300\&dpr=4\&quality=100\&sign=2c382fe4\&sv=2)

Com os passos acima realizados, clicar no botão **Finalizar** ou utilizar a tecla **F2** e será exibida uma mensagem informando o número da pré venda gerada no sistema Dmaster.

![](https://dmaster.gitbook.io/base-de-conhecimento-dmaster/~gitbook/image?url=https%3A%2F%2F981618378-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FaHl55K4HG6CuWy9zUjR9%252Fuploads%252FVgVyaDObgBa1OB7bUO9L%252Fpre%2520venda%2520gravada.PNG%3Falt%3Dmedia%26token%3Db31b13a0-fcb1-4a50-8638-217a2b95b36a\&width=300\&dpr=4\&quality=100\&sign=434aa521\&sv=2)

Caso o produto lançado não tenha estoque suficiente para venda, é exibida uma mensagem ao clicar em **finalizar**, ou seja, é possível fazer o lançamento do mesmo normalmente, mas ao tentar finalizar a venda, a mesma não será concluída.

![](https://dmaster.gitbook.io/base-de-conhecimento-dmaster/~gitbook/image?url=https%3A%2F%2F981618378-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FaHl55K4HG6CuWy9zUjR9%252Fuploads%252FHYQ7R1Sxw8rCq5GNlaIO%252FEstoque.PNG%3Falt%3Dmedia%26token%3Dc9633f84-5dcb-41f9-92d7-c0e4212520c4\&width=300\&dpr=4\&quality=100\&sign=4500b871\&sv=2)
