# Oque seriam as movimentações?

Para desintegrar o sistema, não preciso de nenhum requisito. O sistema pode ser desintegrado a qualquer momento. O que trava é justamente na hora de integrar.

Onde ja temos requisitos a serem seguidos.

{% hint style="info" %}
<mark style="color:$danger;">Situação do envio das movimentações:</mark> \ <mark style="color:$danger;">O sistema considera a "movimentação do dia" a data informada pelo atendente no campo "data da prescrição" ou no campo "data do recebimento" no caso de entrada de nfe.</mark>&#x20;
{% endhint %}

**Quando falamos que não pode ter nenhuma movimentação, estamos se referindo a ideia de que para realizar a integração, não pode ter anda de entrada e saída, mesmo que**&#x20;

(Havia duas movimentações pendentes de dispensação para que fosse possível realizar o envio da movimentação.

Onde é necessário corrigir do MS 1.2568.0334.003-1 para Antimicrobiano.)

Em contato com a Sheila, foram esclarecidas as dúvidas sobre o procedimento de correção da classe terapêutica. Foi informado que, nesse caso, é necessário desintegrar o sistema, finalizar o inventário, realizar a correção do controlado e então reenviar o inventário, aguardando o aceite da ANVISA, para realizar a integração novamente.

Como a cliente desejava enviar as últimas movimentações, o envio foi realizado na data de hoje. Foi explicado que o recomendado é aguardar o aceite da ANVISA antes de desintegrar o sistema.&#x20;
