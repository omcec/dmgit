# Transferência de produtos

Para realizar a transferência de produtos entre filiais é necessário que o módulo de transferência de produtos do DMASTER FARMA já esteja configurado e a loja que está enviando esteja habilitada para emitir nota fiscal eletrônica com o software DMASTER NFe já instalado e configurado.

1. Selecionar a filial para ser realizada a transferência
2. Tipo de transferência
3. Código de barras ou código interno do produto
4. Quantidade a ser transferida
5. Inclusão do item na transferência
6. Gravar e finalizar

{% hint style="info" %}
Só é permitido realizar transferência entre lojas de produtos controlados pela ANVISA através do tipo **“Por Nota fiscal”**, onde será gerado um registro para geração de NFe no sistema DMASTER NFe.
{% endhint %}
