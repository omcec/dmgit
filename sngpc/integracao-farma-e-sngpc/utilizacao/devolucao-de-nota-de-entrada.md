# Devolução de nota de entrada

O módulo de devolução de entrada de nota é utilizado quando de fato é necessário realizar uma devolução para o fornecedor ou quando foi inserida alguma informação errada no momento da entrada nos dados que são enviados para a ANVISA e esses precisam ser corrigidos pelo responsável técnico. Quando a devolução da entrada é realizada para correção de informação da entrada o próprio DMASTER SNGPC fica responsável de excluir a devolução e ajustar o estoque dos produtos.

1. CNPJ do fornecedor
2. Número da nota
3. Botão para pesquisar a nota
4. Quantidade já devolvida para aquele item
5. Botão para informar a quantidade de devolução

Após informar a quantidade de devolução do produto, será exibida a tela para que sejam incluídas as informações do produto.

1. Registro MS
2. Lote
3. Quantidade do produto na nota de acordo com o lote
4. Botão para inserir as informações no grid
5. Gravar e finalizar a devolução
