# Venda

**DMASTER FARMA**

Para realizar vendas de produtos controlados pela ANVISA é necessário fazer a pré-venda destes produtos no módulo de pré-vendas do DMASTER FARMA.

Ao lançar o item na tela de vendas ele será identificado com a cor azul e deve ser pressionada a tecla **F10**, selecionar o foco em cima do item ou clicar sobre o item desejado e pressionar **F3** sobre ele para informar os dados que serão enviados para a ANVISA.

Após pressionar a tecla **F3** sobre o produto controlado pela ANVISA o sistema irá exibir a tela abaixo, onde o usuário deverá informar todos os campos que estão habilitados para edição. Informar a quantidade de cada lote que será vendido, clicar em CONFIRMAR e finalizar a venda normalmente.

#### Recuperar dados da receita

Ao realizar o lançamentos de vários produtos controlados que pertencem a mesma receita, é possível realizar o processo de recuperar os dados da receita, para que não seja necessário informar todos os dados da receita novamente. Para isso é preciso seguir os passos:

1. Lançar todos os produtos da receita
2. Selecionar o primeiro produto e utilizar a tecla F3 para incluir os dados da receita
3. Informar todos os dados necessários e clicar em confirmar
4. Selecionar o próximo produto e utilizar a tecla F3
5. Na tela de dispensação de medicamentos, utilizar a tecla F5 para que o sistema recupere os dados da receita informados no produto anterior

{% hint style="info" %}
Ao realizar uma pré-venda de produto controlado, o estoque é baixado no DMASTER FARMA e no DMASTER SNGPC e o registro da venda já constará no módulo de confirmação de lançamentos do DMASTER SNGPC.\
Se a pré-venda for excluída, o registro também será excluído do SNGPC.
{% endhint %}

#### **Venda realizada com produto não classificado**

Caso seja realizada venda de produto controlado, mas o mesmo ainda não foi classificado, será necessário seguir os passos:

1. Realizar a devolução da venda no DMASTER FARMA
2. Classificar o produto corretamente no DMASTER FARMA
3. Realizar a venda novamente

#### Venda de produto que possui mais de um Registro MS

Para produtos que possuem mais de um registro MS no cadastro, ao lançá-lo na tela de pré-vendas do DMASTER FARMA, o sistema irá exibir a tela abaixo para que o usuário selecione o registro MS desejado.

#### Impressão dos dados da receita

É possível realizar a impressão dos dados da receita através da tela de pré-vendas, de acordo com impressao-dos-dados-da-receita.

#### **DMASTER SNGPC**

O módulo de dispensação no sistema DMASTER SNGPC só é utilizado quando for necessário realizar alguma correção em uma venda que não foi inserida os dados que são enviados para a ANVISA ou não foi possível alterar os dados inseridos no momento da venda na tela de Confirmação de lançamentos.\
Para realizar esta correção é necessário antes realizar a devolução dos itens a serem corrigidos no módulo de Devolução de venda no DMASTER FARMA e posteriormente fazer o lançamento manual no módulo de dispensação abaixo, informando o número da venda para o sistema fazer as devidas validações.
