# Devolução de transferência

Para realizar uma devolução de transferência é necessário recupera-la através do número do documento e clicar em cancelar. Para isso o sistema irá validar se já existe NFe autorizada para o documento informado, caso exista deverá ser feito o cancelamento no sistema DMASTER NFe.
