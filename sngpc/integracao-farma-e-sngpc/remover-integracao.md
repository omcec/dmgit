# Remover integração

{% hint style="danger" %}
Antes de iniciar o processo de remoção da integração, deve ser conferido o módulo de confirmação de lançamentos no SNGPC. Não pode haver nenhum registro pendente de confirmação.\
Caso haja algum registro pendente, este deve ser confirmado para depois ser iniciado o processo de remoção da integração.
{% endhint %}

Para remover a integração FARMA e SNGPC, é preciso acessar as configurações gerais do sistema no DMASTER FARMA (Shift + F12). Após acessar o módulo, **desabilitar** o parâmetro **Sistema SNGPC Integrado.**

Após desabilitar o parâmetro, reiniciar todos os sistemas em todos os computadores do estabelecimento.
