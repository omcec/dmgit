# Inventário Inicial

Este módulo é utilizado para realizar o cadastro dos medicamentos controlados que possui no estabelecimento ou realizar correção de informação quando o inventário é finalizado.

#### Cadastro de medicamento

Para realizar o cadastro do medicamento corretamente, os seguintes campos devem ser preenchidos:

1. **Registro MS:** informar o registro MS do medicamento;
2. **Classe terapêutica:** selecionar a classe terapêutica do medicamento (antimicrobiano ou controle especial)
3. **CID (Classificação Internacional de Doenças):** este código não é necessário ser informado;
4. **Quant. total:** quantidade total em estoque do medicamento, independente do lote;
5. **Descrição / Apresentação / Fabricante:** informar a descrição do produto da melhor forma que o Responsável Técnico deseja, incluindo descrição, apresentação e fabricante do produto. Exemplo: Zolpidem 25MG 30CP CIMED;
6. **Tipo de receituário:** selecionar o tipo de receituário do medicamento ( branca 2 vias, notificação B azul, notificação especial branca, notificação A amarela, antimicrobiano 2 vias e notificação B2 azul);
7. **Código DCB (Denominação Comum Brasileira):** é possível informar apenas o código 1, ou pesquisar qual o código correto do medicamento;
8. **Descrição DCB:** pesquisar a descrição correta de acordo com o código DCB, ou caso seja utilizado o código 1, o campo poderá ficar vazio;
9. **Lote:** informar o lote do medicamento;
10. **Quantidade:** informar a quantidade em estoque do lote em questão;
11. **Unidade:** selecionar a unidade (CX ou FR);
12. **Validade:** informar a validade do lote.

Após preencher todas os campos corretamente, é necessário clicar em gravar para concluir o processo.

#### Correção de informação no inventário

O módulo também pode ser utilizado para corrigir informações dos medicamentos quando é realizada a finalização de inventário. Nesse caso, ao informar o registro MS ou descrição do produto, seus dados cadastrais serão carregados automaticamente se ele já possuir cadastro no sistema. Com as informações carregadas, os ajustes podem ser realizados normalmente.\
As correções poderão ser realizadas nas informações de: classe terapêutica, estoque, descrição, tipo de receituário, lote, unidade e validade.\
Ao inserir o Registro MS ou descrição do produto já cadastrado que possui estoque no sistema, será mostrada a mensagem abaixo.

A pesquisa de produto pode ser feita por Registro MS ou descrição.
{% endhint %}

{% hint style="info" %}
No campo código DCB deve ser inserido o código 01, para que não fique sem informação e gere erros no sistema ou deve ser realizada a pesquisa do código DCB correto e inserir o mesmo.
{% endhint %}
