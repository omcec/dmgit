# Lançamentos

Módulo utilizado para realizar consulta e exclusão de movimentações de entrada e saída que foram realizadas.

<figure><img src="https://981618378-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FaHl55K4HG6CuWy9zUjR9%2Fuploads%2FTD87GrBlNOWANVoIZJTf%2FLancamentos.PNG?alt=media&#x26;token=c0a19066-9316-49d4-89c5-f78343fd0091" alt=""><figcaption></figcaption></figure>

### Consultar movimentação

Para realizar a consulta de movimentação é preciso selecionar o tipo de lançamento (todos, entradas, saída, perda, transferência) e informar o período desejado.\
É possível utilizar filtros, como: número da nota fiscal de entrada e Registro MS do produto. Dessa forma, as movimentações apresentadas serão de acordo com o filtro selecionado.

<figure><img src="https://981618378-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FaHl55K4HG6CuWy9zUjR9%2Fuploads%2FKIHT0Q3WgIVKvvVlDwSi%2FLancamentos%201.PNG?alt=media&#x26;token=cb6d0198-dc7b-43b0-a62c-8e83367ce1ba" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
Ao selecionar o tipo de lançamento todos, serão exibidas todas as movimentações realizadas no sistema, independente do tipo.
{% endhint %}

### Relatório de movimentação

É possível visualizar os registros de movimentação através de relatório. Para isso, é utilizado o botão Imprimir, após selecionar os filtros desejados.

<figure><img src="https://981618378-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FaHl55K4HG6CuWy9zUjR9%2Fuploads%2F71uBMnXz5PfJCbU5pdva%2FImprimir%20relatorio%20movimentacoes.png?alt=media&#x26;token=28a7d399-3426-4dca-b785-9f9450c6ccc8" alt=""><figcaption></figcaption></figure>

O relatório será gerado com todas as movimentações separadas por data e tipo (entrada e saída).

<figure><img src="https://981618378-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FaHl55K4HG6CuWy9zUjR9%2Fuploads%2FwX24m2zV3Q8lzHaUy34K%2FRelatorio%20de%20lancamentos.PNG?alt=media&#x26;token=63b665ca-61de-409b-b5b3-9457022af928" alt=""><figcaption></figcaption></figure>

### Excluir movimentação

Caso seja realizada uma movimentação de entrada ou saída incorretamente, é possível que esta seja excluída. Para isso, é preciso buscar a movimentação, selecioná-la e clicar em excluir.

<figure><img src="https://981618378-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FaHl55K4HG6CuWy9zUjR9%2Fuploads%2FHx2n0rUXHjm4MJl2UWlu%2Fexcluir%20movimentacao.png?alt=media&#x26;token=3acb1f9d-84c7-4b2a-befa-39d0e734e8ab" alt=""><figcaption></figcaption></figure>

Após o processo anterior, será solicitada a senha de administrador para conclusão do processo.

<figure><img src="https://981618378-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FaHl55K4HG6CuWy9zUjR9%2Fuploads%2FJJRCaE3LBNZB8pb0zbrS%2FSenha%20adm%20confirmar%20inventario.PNG?alt=media&#x26;token=e070a9a9-34e8-42fd-84a7-67b6bca03d10" alt=""><figcaption></figcaption></figure>

{% hint style="danger" %}
Só é possível excluir movimentações que ainda não foram enviadas para a ANVISA.
{% endhint %}
