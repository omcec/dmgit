# Entradas

Módulo utilizado para que seja realizada a entrada de produtos controlados no DMASTER SNGPC.\
Para realizar o processo, é preciso preencher os seguintes campos:

1. **Nº NF:** inserir o número da nota de entrada;
2. **CNPJ:** inserir o CNPJ do fornecedor;
3. **Tipo de operação:** selecionar o tipo de operação da nota (compra ou transferência);
4. **Fornecedor:** será preenchido automaticamente com o nome do fornecedor ao ser informado o CNPJ do mesmo;
5. **Emissão:** selecionar a data de emissão da nota;
6. **Recebimento:** selecionar a data de recebimento da nota:
7. **Código MS:** informar o código MS do produto;
8. **CID (Classificação Internacional de Doenças):** código que não é obrigatório ser preenchido;
9. **Classe Terapêutica:** selecionar qual a classe terapêutica do medicamento (antimicrobiano ou controlado); **\*Caso o produto já esteja cadastrado, ao inserir o Código MS do produto, este campo já é preenchido automaticamente, de acordo com o cadastro.**
10. **Tipo de Receituário:** selecionar o tipo de receituário do medicamento ( branca 2 vias, notificação B azul, notificação especial branca, notificação A amarela, antimicrobiano 2 vias e notificação B2 azul);**\*Caso o produto já esteja cadastrado, ao inserir o Código MS do produto, este campo já é preenchido automaticamente, de acordo com o cadastro.**
11. **Descrição/Apresentação/Fabricante:** informar a descrição do produto da melhor forma que o Responsável Técnico deseja, incluindo descrição, apresentação e fabricante do produto. **Exemplo:** Zolpidem 25MG 30CP CIMED; **\*Caso o produto já esteja cadastrado, ao inserir o Código MS do produto, este campo já é preenchido automaticamente, de acordo com o cadastro.**
12. **Unidade:** selecionar a unidade do medicamento (CX ou FR);
13. **Quant. Total:** informar a quantidade total do medicamento presente na nota em questão, independente do lote;
14. **Código DBC (Denominação Comum Brasileira):** é possível informar apenas o código 1, ou pesquisar qual o código correto do medicamento;
15. **Descrição DCB:** pesquisar a descrição correta de acordo com o código DCB, ou caso seja utilizado o código 1, o campo poderá ficar vazio;
16. **Lote:** informar o lote do medicamento;
17. **Quant. Lote:** informar a quantidade do lote em questão;
18. **Validade:** informar a validade do lote.
19. **Ok:** finalizar o lançamento do produto e incluir o mesmo no grid;
20. **Gravar:** finalizar e gravar a entrada da nota.

{% hint style="info" %}
O fornecedor deve ser cadastrado no DMASTER FARMA para que o DMASTER SNGPC consiga preencher as informações dele automaticamente na entrada de nota.
{% endhint %}
