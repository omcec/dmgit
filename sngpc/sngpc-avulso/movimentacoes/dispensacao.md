# Dispensação

<figure><img src="https://981618378-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FaHl55K4HG6CuWy9zUjR9%2Fuploads%2FdX9Ck34TBAGNjS8bMrki%2FMenus%20sngpc%20dispensacao.jpg?alt=media&#x26;token=504b6839-d933-4ca9-9d15-e77586bb6fb3" alt=""><figcaption></figcaption></figure>

Módulo utilizado para que seja realizada a dispensação dos medicamentos controlados no DMASTER SNGPC.\
Para realizar o processo, é preciso preencher os seguintes campos:

1. **Tipo de receituário:** selecionar o tipo de receituário do medicamento;
2. **Nº da receita:** informar o número da receita; \***Receita Branca 2 vias e Receita Antimicrobiano 2 vias, não é necessário informar o número, este é preenchido automaticamente pelo sistema.**
3. **Data Prescrição:** informar a data de prescrição da receita;
4. **Reg. Profissional:** informar o registro do profissional;
5. **Conselho Prof.:** informar o conselho do profissional;
6. **UF Conselho:** selecionar UF do conselho do profissional;
7. **Nome do Prescritor:** informar o nome do prescritor;
8. **Uso medicamento:** campo preenchido automaticamente de acordo com o conselho do profissional (Humano ou Veterinário);
9. **Doc. Comprador:** informar o número do documento do comprador;
10. **Tipo de Documento:** selecionar qual tipo de documento foi informado;
11. **Órgão Expedidor:** selecionar o órgão expedidor do documento informado;
12. **UF Doc:** selecionar a UF do documento informado;
13. **Nome do Comprador:** informar o nome do comprador;
14. **Nome do paciente:** informar o nome do paciente; **\*informação utilizada apenas para receitas de antimicrobiano.**
15. **Sexo do paciente:** selecionar o sexo do paciente; **\*informação utilizada apenas para receitas de antimicrobiano.**
16. **Idade (anos):** informar a idade do paciente; **\*informação utilizada apenas para receitas de antimicrobiano.**
17. **Data da venda:** selecionar a data de realização da venda;
18. **Registro MS:** informar o registro MS do produto;
19. **Descrição:** após o preenchimento do registro MS do produto, o campo descrição é preenchido automaticamente;
20. **Unidade:** após informar o registro MS do produto, o campo é preenchido automaticamente;
21. **CID:** caso esteja preenchido no cadastro do produto, o campo é preenchido automaticamente após informar o registro MS do produto;
22. **Uso Prolongado:** selecionar se o medicamento será de uso prolongado;
23. **Qt. Prescrita:** informar qual a quantidade prescrita do medicamento de acordo com a receita;
24. **Ok:** enviar o medicamento para o grid;

<figure><img src="https://981618378-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FaHl55K4HG6CuWy9zUjR9%2Fuploads%2FJLPbQmmS4OGtr9zGs7FF%2FDispensacao.JPG?alt=media&#x26;token=e2b3ce41-e63d-4ad0-b867-1560dace17ea" alt=""><figcaption></figcaption></figure>

Após preencher todos os campos corretamente, será mostrado no grid o produto e seus respectivos lotes, sendo necessário inserir a quantidade prescrita no lote correto. Com a quantidade informada, clicar em Ok para gravar a dispensação.

<figure><img src="https://981618378-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FaHl55K4HG6CuWy9zUjR9%2Fuploads%2FF7mE1IdabpL9dRi01DgV%2FDispensacao%20medicamento.JPG?alt=media&#x26;token=5417f01f-84ac-464a-9484-f84a51911a47" alt=""><figcaption></figcaption></figure>

#### Excluir medicamento

Caso seja necessário excluir um medicamento do lançamento da receita, é preciso selecionar o mesmo e utilizar a tecla Delete.

<figure><img src="https://981618378-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FaHl55K4HG6CuWy9zUjR9%2Fuploads%2FIwNDuxltjOXcoFA90qpy%2FExcluir%20medicamento.JPG?alt=media&#x26;token=068c56be-de21-418c-81dd-878fe314b5b0" alt=""><figcaption></figcaption></figure>

Após clicar em ok para finalizar o processo de dispensação, é apresentada a mensagem de alerta para realizar o cadastro do médico e do cliente pertencente aquela venda, onde o usuário pode escolher se deseja ou não realizar o cadastro. Ao confirmar a mensagem, o sistema irá realizar o cadastro automaticamente de acordo com as informações que já foram inseridas. Sendo assim, será possível pesquisar aquele cliente ou médico em dispensações posteriores.

#### Pesquisa

É possível realizar a pesquisa de prescritor já cadastrado, através do campo Reg. Profissional. Para isso, é preciso clicar no campo, utilizar a tecla F2 e pesquisar o nome do profissional. Ao encontrar o prescritor desejado, selecioná-lo e utilizar a tecla enter para que os dados do mesmo sejam carregados.

<figure><img src="https://981618378-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FaHl55K4HG6CuWy9zUjR9%2Fuploads%2Fgvyhxzv3qL2Gh7QTqZxk%2FPesquisar%20prescritor.JPG?alt=media&#x26;token=42787433-9990-4a38-871e-50fab378bcd3" alt=""><figcaption></figcaption></figure>

A pesquisa também pode ser feita por comprador, utilizando o campo Doc. Comprador. Para isso, é preciso clicar no campo, utilizar a tecla F2 e pesquisar o nome do comprador. Ao encontrar o comprador desejado, selecioná-lo e utilizar a tecla Enter para que os dados do mesmo sejam carregados.

<figure><img src="https://981618378-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FaHl55K4HG6CuWy9zUjR9%2Fuploads%2F6lCAOhlUX3rNgWVFHXQa%2FPesquisa%20comprador.JPG?alt=media&#x26;token=1543cc6c-5970-4169-addf-4240b5a95e23" alt=""><figcaption></figcaption></figure>

#### Excluir prescritor

É possível também realizar a exclusão do cadastro do prescritor. Para isso, é preciso buscar o prescritor, selecioná-lo para carregar os dados, utilizar a tecla F10 e clicar em sim na mensagem de confirmação para finalizar o processo.

<figure><img src="https://981618378-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FaHl55K4HG6CuWy9zUjR9%2Fuploads%2FGJbuUzfyeHkFTlR1iun2%2FExclusao%20de%20prescritor.JPG?alt=media&#x26;token=e570736c-47e4-4f9c-a9f1-3451b81175b5" alt=""><figcaption></figcaption></figure>

#### Excluir cliente

Para realizar a exclusão de cliente já cadastrado é preciso buscar o mesmo, para que as informações sejam carregadas, utilizar a tecla F12 e clicar em sim na mensagem de confirmação para finalizar o processo.
