# Acessando o Sistema

Primeiro acesso

Será exibido uma tela onde deverão ser inseridos os dados do estabelecimento e os dados do Responsável Técnico, será necessário informar também a senha Administrador do sistema.

#### Próximos acessos

Ao acessar o Sistema será solicitado a senha "Administrador", senha cadastrada pelo "Responsável Técnico" no momento que o mesmo foi cadastro no sistema.

![](https://981618378-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FaHl55K4HG6CuWy9zUjR9%2Fuploads%2FYXrD7R7o2TgyblQ3E4jb%2Fimage.png?alt=media\&token=849b952e-c66d-4a35-b840-a64301741ab0)

Após ser inserido a senha do Administrador será solicitado o Usuário e Senha, usuário que é utilizado no DMASTER FARMA.
