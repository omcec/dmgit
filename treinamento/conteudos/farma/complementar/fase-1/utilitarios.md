# Utilitários

## Utilitários

<details>

<summary>ETIQUETAS</summary>

1. Este módulo serve para realizar impressão de etiquetas dos produtos.
2. É possível carregar os produtos direto do cadastro ou de uma nota fiscal que você já tenha dado entrada.
   * Preço: além do preço do produto possui o código interno. Geralmente branca para colocar direto no produto.
   * Gôndola: além do preço ela também tem o código de barras ou código interno. Geralmente amarela para colocar na prateleira.

</details>

<details>

<summary><mark style="color:red;">*</mark> PROGRAMAÇÃO DE OFERTAS</summary>

1. Este módulo serve para criar ofertas por um período específico.
   * Você insere o produto desejado.
   * Informa desconto pela % ou pelo preço promocional.
   * Informa a data de início e fim e clica em OK.
   * Se quiser, pode manter o preço com desconto anterior a oferta programada. Dessa maneira, ao fim da oferta programada, o preço com desconto retorna para o valor anterior.
   * Também é possível prorrogar o prazo.
     * <mark style="color:blue;">Caso a oferta seja programada para o dia corrente, o sistema deve ser reiniciado para que a oferta fique vigente.</mark>

</details>

<details>

<summary>MANUTENÇÃO DE PRODUTOS</summary>

1. Este módulo serve para fazer ajuste de preço (para mais ou para menos), ajuste de comissão ou aplicar preço com desconto em diversos produtos ao mesmo tempo. É possível também alterar descrição, fabricante e classe.
   * No campo **Código**, você consegue carregar apenas um produto;
   * Você pode filtrar por **Classe** para carregar todos os produtos da classe;
   * Você pode filtrar por **Fabricante**;
   * **Consulta Geral**: serve para incluir os produtos inativos;
   * Filtrar por Situação: todos, Monitorados (produtos ABCFarma), Livres;
   * Após clicar em Localizar, os produtos serão carregados.
2. Com isso se for ajuste de valor, basta:
   * Inserir o valor no campo Valor.
   * Marcar se o valor será em reais ou porcentagem.
   * E o tipo: se será reajuste positivo, negativo, promoção ou comissão.
   * Com isso, basta clicar Aplicar.
     * <mark style="color:blue;">As alterações realizadas são aplicadas para todos os itens que estiverem no grid, exceto quando a alteração é realizada diretamente no item.</mark>

</details>

<details>

<summary>INVENTÁRIO</summary>

1. Este módulo serve para atualizar a quantidade dos produtos no estoque.
2. O inventário serve para deixar seu estoque organizado dentro do sistema.
3. Você pode demorar o tempo que precisar, não precisa terminar no mesmo dia.
4. Pode ser feito com mais de um usuário.
5. No término pode zerar e inativar os produtos não contados.
6. É possível fazer o inventário parcial ou por classe.
   * ​No **Inventário Parcial**, o sistema não irá dar a opção de **Zerar e Inativar** os não contados.

</details>

<details>

<summary>ATUALIZAR</summary>

1. A maioria é para uso da DMaster, mas pode acontecer de usarmos:
   * **Versão do Sistema;**
     * Quando uma versão nova está liberada, o sistema no Servidor já pergunta se você quer atualizar. Caso você ignore a mensagem, é possível atualizar neste botão.
     * Após a atualização no Servidor, basta fechar e reabrir nos Terminais que a atualização ocorrerá automaticamente.
   * **Produtos E-Pharma:** para gerar a tabela de preços para os produtos do PBM;​​​​​​​

</details>

<details>

<summary>ATUALIZAÇÃO DE PREÇOS</summary>

1. **Lista Geral de Preços**: serve para atualização de preços dos produtos da ABCFarma;
2. **Assinantes ABCFarma**: serve para os assinantes da ABCFarma;
3. **Distribuidora Santa Cruz**: serve para atualizar os preços conforme tabela da Santa Cruz.

</details>

<details>

<summary>ALTERNAR ENTRE FILIAIS</summary>

* Serve para você ter acesso aos dados gerenciais de uma filial.

</details>

<details>

<summary>PARÂMETROS FILIAIS</summary>

1. Serve para configurar o acesso remoto para ter acesso ao sistema fora da rede da loja. De casa, por exemplo;
2. E para efetuar cadastro das lojas filiais.

</details>

<details>

<summary>EMPRESA</summary>

1. **Atualizar dados**: caso você já tenha solicitado alguma atualização cadastral e no sistema ainda não estiver recebido a informação nova;
2. **Visualizar dados**; dados da empresa;
3. **Certificado Digital**: para importação de Certificado Digital para atualização.

</details>

<details>

<summary><mark style="color:red;">*</mark> TRIBUTAÇÃO</summary>

1. **Gesto tributário**: é um serviço com mensalidade adicional;
   * Ele atualiza os códigos tributários dos cadastros dos produtos para validação correta na SEFAZ no momento da venda.
   * <mark style="color:blue;">Caso o cliente demonstre interesse, pode ser aberto um chamado para o setor comercial no final do treinamento.</mark>
2. **Exportação de produtos**: se não quiser contratar o serviço, pode baixar uma planilha com os seus produtos, efetuar a correção e importar novamente com os dados atualizados.

</details>

<details>

<summary>DIGITALIZAR DOCUMENTOS</summary>

* Para uso exclusivo no gerenciamento de documentos e receitas de clientes que compram produtos da Farmácia Popular.

</details>

<details>

<summary><mark style="color:red;">*</mark> ASSINATURA DO SISTEMA</summary>

* Para baixar boletos em aberto de assinatura do sistema DMaster:
  * O boleto é disponibilizado até o segundo dia útil do mês com o vencimento para todo dia 10.

</details>

<details>

<summary>INFORMAÇÕES IMPORTANTES</summary>

1. Serve para reabrir os banners que o sistema mostra ao ser iniciado;
2. Aqui exibirá informações e comunicados importantes sobre o sistema e a DMaster.

</details>
