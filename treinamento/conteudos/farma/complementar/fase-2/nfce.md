# NFCE

## NFCE

<details>

<summary><mark style="color:red;">*</mark> MANUTENÇÃO DO NFCe</summary>

* **Cancelamento de NFCe em até 30min**:
  * Caso ultrapasse o tempo, o cliente poderá solicitar a contabilidade ou emitir uma NFe para cancelar o cupom.

- **Reimpressão de NFCe**: pode ser impresso a qualquer momento.

* Informar sobre o campo de manutenção das vendas com erro:
  * Conferir no campo Mensagem o que precisa ser corrigido;
  * Conferir detalhes da venda **+** Acessar cadastro do produto **+** Corrigir informações do produto;
  * Reenviar venda, exceto PBMs. que deve ser feito cancelamento e a venda refeita.

- **Limite de 50 vendas com erro**.
  * Como as vendas são validadas online, quando há uma recusa da SEFAZ, o sistema já informa que você deve acessar o Módulo de Manutenção de NFCe para corrigir.
  * A partir de 10 vendas pendentes de correção, o sistema já o notifica alertando para correção.
  * Após a correção do erro, é possível o reenvio, exceto venda de PBM.

* **Relatório NFCe**:
  * Relatório de uso da contabilidade contendo os registros de cada venda efetuada e arquivos XMLs de cada venda.
  * A emissão pode ser feita a partir do 1º dia de cada mês.

</details>
