# SNGPC

{% hint style="danger" %}
**Responsável Técnico (RT)**

O Farmacêutico deve estar associado junto a Anvisa como responsável técnico do estabelecimento para utilização do DMASTER SNGPC.
{% endhint %}

#### Acesso ao sistema

Ao acessar o sistema pela primeira vez é solicitado o cadastro da empresa e do Responsável Técnico do estabelecimento. Caso o cadastro já tenha sido realizado anteriormente, ao acessar o sistema, é solicitada a senha de administrador, criada pelo próprio RT e logo após é solicitado usuário e senha cadastrado anteriormente no DMASTER FARMA.

{% hint style="info" %}
A senha do Administrador é cadastrada no momento do cadastro da empresa.
{% endhint %}

{% hint style="info" %}
**Pesquisa por MS**

A pesquisa de produto por Registro MS nesse módulo deve conter os pontos.
{% endhint %}

{% hint style="info" %}
**Alteração MS e Classe terapêutica**

Registro MS e classe terapêutica só podem ser alterados com o inventário finalizado.
{% endhint %}

{% hint style="info" %}
**Envio de inventário**

Caso o inventário enviado tenha sido rejeitado pela Anvisa e seja corrigido no mesmo dia do envio, não será possível enviá-lo nesse mesmo dia, pois só é possível enviar um arquivo por dia.
{% endhint %}

{% hint style="info" %}
**Envio de Insumos**

Verificar se o cliente trabalha com Insumos, caso trabalhe o mesmo terá que ser enviado juntamente com o inventário dos Medicamentos.
{% endhint %}

{% hint style="info" %}
**Confirmação de inventário**

O inventário só pode ser confirmado no sistema DMASTER SNGPC um dia após a validação da Anvisa. Não é possível confirmá-lo no mesmo dia da validação.
{% endhint %}

{% hint style="info" %}
**Finalização de inventário**

No processo de finalização do inventário, o sistema permite que o RT escolha entre manter o estoque dos produtos cadastrados ou zerar. Se zerar, apenas o estoque é zerado, os produtos continuam cadastrados no sistema. Se manter, o cadastro e estoque dos produtos são mantidos.
{% endhint %}

#### Transmitir Movimentação

* [ ] Explicar sobre o módulo.
* [ ] Explicar como verificar o status dos arquivos enviados.
* [ ] Explicar como cancelar arquivo enviado.

{% hint style="info" %}
**Envio de arquivo**

* Arquivos enviados devem conter movimentações de um período de no mínimo 1 (um) a no máximo 7 (sete) dias.
* Só é possível realizar um envio de arquivo por dia.
* Mesmo que não haja movimentação, o arquivo daquele dia precisa ser enviado para a Anvisa.
* A senha informada para envio de arquivo, precisa conter os 06 primeiros dígitos da senha utilizada para acessar o site da Anvisa.
* Após enviar arquivo para a Anvisa, quando ele é validado, ao abrir o sistema, é exibido um alerta informando a situação do arquivo, se foi aceito ou rejeitado.
{% endhint %}
