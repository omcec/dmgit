# Informações iniciais

## Informações iniciais

{% hint style="warning" %}
**Obrigatoriedade nos conteúdos**

Em qualquer momento da documentação referente aos treinamentos, os tópicos que estivem com o caractere ( <mark style="color:red;">\*</mark> ) no título indica que aquele assunto é de extrema importância de ser explicado ao usuário. Os tópicos que não possuem esse flag devem ser explicados apenas superficialmente e caso o usuário se interesse ou faça sentido para o seu estabelecimento, deve ser detalhado.
{% endhint %}

#### **Antes de iniciar as etapas, informar sobre:** <a href="#antes-de-iniciar-as-etapas-informar-sobre" id="antes-de-iniciar-as-etapas-informar-sobre"></a>

#### <mark style="color:red;">\*</mark> Caso o cliente possua mais de um Terminal

* [x] Terminal Servidor, Terminal Comum (balcão de consulta e pré-venda), Terminal Caixa (Emissor de Cupom Fiscal).
* [x] Informar sobre o Terminal Administrador, caso ele fique em um terminal diferente do servidor.

#### <mark style="color:red;">\*</mark> Backup

* [x] Informar que o sistema salva uma cópia automaticamente a cada hora em um terminal diferente do servidor.
* [x] Informar sobre o backup manual caso o cliente tenha apenas um computador.

#### <mark style="color:red;">\*</mark> Comandos básicos

* [x] **ESC**, para abandonar a tela.
* [x] **Enter**, para confirmar alteração em um determinado campo, para selecionar ou ir para o próximo campo.
* [x] **Campo código**, utilizado para pesquisa.
