# Checklist instalação

A instalação de Servidor deve ser realizada seguindo um passo a passo. Com isso, serão citados os processos a serem realizados, para que a instalação seja concluída com sucesso. Verifique a explicação detalhada do processo de instalação em [Servidor](https://dmaster.gitbook.io/base-de-conhecimento-dmaster/documentacao-interna/documentacao-tecnica/instalacoes/servidor) .

**Configurações do computador**

* [ ] Segurança do disco local C:
* [ ] Configurações de rede e compartilhamento
* [ ] Renomear computador

**Cópia/baixa de arquivos**

* [ ] Copiar pastas dos sistemas que serão instalados de acordo com necessidade do cliente
* [ ] Copiar executáveis dos sistemas que serão instalados de acordo com necessidade do cliente
* [ ] Baixar instalador de servidor

**Antivírus**

* [ ] Criar exceção no antivírus incluindo todas as pastas dos sistemas DMASTER.

**Instalações**

* [ ] Instalar o setup servidor
* [ ] Instalar sistemas de acordo com necessidade do cliente

**Banco de dados**

* [ ] Anexar banco de dados

**Dados da empresa**

* [ ] Exportar dados da empresa
* [ ] Importar dados da empresa

**Configurações do sistema**

* [ ] Configurar backup
* [ ] Ajustar Config.ini

**Cadastro de terminal administrador**

* [ ] Cadastrar novo terminal administrador

**Abertura dos sistemas**

* [ ] Executar todos os sistemas instalados.

Após finalizar a instalação, verificar se ainda é necessário a instalação/configuração de outros aplicativos e utilitários do sistema, de acordo com as documentações [Checklist DMASTER FARMA](https://dmaster.gitbook.io/base-de-conhecimento-dmaster/documentacao-interna/documentacao-tecnica/instalacoes/servidor/checklist-dmaster-farma) e [Checklist DMASTER SHOP](https://dmaster.gitbook.io/base-de-conhecimento-dmaster/documentacao-interna/documentacao-tecnica/instalacoes/servidor/checklist-dmaster-shop).
