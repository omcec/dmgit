# Impressora fiscal

#### Impressora não comunica com o computador ou não imprime cupom <a href="#impressora-nao-comunica-com-o-computador-ou-nao-imprime-cupom" id="impressora-nao-comunica-com-o-computador-ou-nao-imprime-cupom"></a>

A não impressão dos cupons fiscais ou falha de comunicação com o computador, pode acontecer por vários motivos. Com isso, serão citados abaixo alguns pontos a serem verificados para o problema ser solucionado. **Caso não seja possível resolver o erro após realizar todos os passos, uma solicitação de ajuda deve ser aberta para o responsável.**

{% hint style="info" %}
Antes de iniciar a verificação dos pontos para tentativa de solucionar o problema, deve ser perguntado ao cliente se foi impressa uma mensagem automaticamente ao ligar a impressora.
{% endhint %}

* [ ] Impressora com bobina

* [ ] Impressora sem luz de erro

**Verificações físicas**

* [ ] Verificar tipo de cabo utilizado na conexão da impressora com o computador
