# Padrões de utilização

Os sistemas DMASTER FARMA E DMASTER SHOP possuem alguns padrões de utilização. São eles:

* **Pesquisa:** todo campo código é utilizado para pesquisa, que podem ser realizadas por descrição/nome, código interno/código de barras;
* **Confirmação/Seleção:** a tecla Enter é utilizada para selecionar o registro pesquisado e carregar os dados na tela;
* **Abandono de tela:** para sair de uma tela e voltar a anterior, é utilizada a tecla Esc;
* **Alternar campo:** para mudar o foco para o campo seguinte, pode ser usado o mouse, para clicar no campo desejado, ou pode ser feito utilizando a tecla Enter;
* **Marcar e desmarcar:** para marcar e desmarcar parâmetros ou registros, são utilizadas as teclas ( + ) e ( - ) do teclado numérico;
* **Abertura do sistema:** O sistema DMASTER FARMA/SHOP deve ser aberto primeiro no Terminal administrador e em seguida no servidor e demais terminais comuns. É obrigatório iniciar primeiro o DMASTER FARMA/SHOP e depois iniciar o DMASTER ECF e DMASTER SNGPC. Ao abrir o DMASTER FARMA/SHOP, os atalhos de todos os sistemas instalados são criados na área de trabalho.

{% hint style="warning" %}
Os sistemas DMASTER não necessitam de internet para o funcionamento, precisam apenas da rede interna funcionando corretamente, para que o sistema de todos os computadores da loja funcionem. Apenas o Servidor funciona corretamente sem rede. Os demais computadores precisam de rede para se comunicarem com o Servidor.

A internet é essencial apenas para alguns processos específicos, como venda de NFCe e venda de farmácia popular.
{% endhint %}
