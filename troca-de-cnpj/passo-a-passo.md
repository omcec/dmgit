# Passo a passo

#### Cadastre-se <a href="#cadastre-se" id="cadastre-se"></a>

Para realizar a troca do CNPJ, é preciso acessar o site da DMASTER - [https://dmaster.com.br/](https://dmaster.com.br/)

Esta funcionalidade é utilizada pelos clientes DMASTER quando vão trocar de CNPJ, ou realizar qualquer operação que seja necessário cadastrar um novo estabelecimento como cliente DMASTER.

Utilizando esta opção, o cliente deve preencher todas as informações corretamente e após clicar em cadastrar, as informações são enviadas para o controle interno DMASTER, ou seja, esse novo estabelecimento é cadastrado no controle.

Caso a cliente abre chamado para o setor de suporte, iremos orientá-la a realizar o cadastro no site conforme o passo anterior, informamos que iremos repassar o chamado para o setor responsável (Titulo no chamado: Alteração de Cadastro; Enviar para o Jairo), informando que deve aguardar o contato do setor para dar andamento as novas tratativas.

{% hint style="danger" %}
Obs: Informar que como é um novo contrato, deve retirar todos os relatórios que precisar, de convênio, justamente por que será impossível retirar posteriormente a troca do CNPJ.

Como se trata de um novo contrato com troca de CNPJ, é importante ressaltar que _todos os relatórios necessários relacionados ao convênio atual devem ser retirados antes da mudança_. Após a alteração do CNPJ, não será mais possível acessar ou gerar esses relatórios.
{% endhint %}
